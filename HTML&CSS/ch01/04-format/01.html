Cau truc HTML tag

Để định dạng cho thẻ HTML ta có những cách sau
  • * có tác dụng đối với tất cả các thẻ HTML
  • Selector
  • Class sẽ mạnh hơn phần định dạng của Selector
  • id
  • Style phần định dạng style mạnh hơn phần định dạng ID
    • Độ ưu tiên của định dạng css: style > id > class > selector > *
